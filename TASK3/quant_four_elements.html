量化交易策略四大要素详解

进场 · 出场 · 仓位 · 风控 — 概念、公式、代码

进场 Entry 出场 Exit 仓位 Position 风控 Risk

一、进场 Entry — 何时买入？

进场规则决定了策略"开仓"的时机。一个好的进场信号应该具备可量化、可回溯、低延迟三个特征。以下是最常用的五类进场指标。

1.1 均线交叉 MA Crossover

概念：利用短周期均线与长周期均线的相对位置判断趋势方向。短线上穿长线为金叉（买入），下穿为死叉（卖出）。这是最经典的趋势跟踪信号。

\[ \text{SMA}_n(t) = \frac{1}{n}\sum_{i=0}^{n-1} P_{t-i} \] \[ \text{EMA}_n(t) = \alpha \cdot P_t + (1-\alpha) \cdot \text{EMA}_n(t-1), \quad \alpha = \frac{2}{n+1} \] \[ \text{Signal} = \begin{cases} +1\ (\text{金叉}), & \text{MA}_{\text{fast}}(t) > \text{MA}_{\text{slow}}(t) \text{ 且昨日 } \le \\ -1\ (\text{死叉}), & \text{MA}_{\text{fast}}(t) < \text{MA}_{\text{slow}}(t) \text{ 且昨日 } \ge \end{cases} \]

原理 EMA 对近期价格赋予更高权重，比 SMA 对趋势变化更敏感。常用组合：5/20（短线）、10/60（中线）、20/120（长线）。

# 双均线交叉信号 import pandas as pd df['ma_fast'] = df['close'].rolling(10).mean() df['ma_slow'] = df['close'].rolling(60).mean() df['signal'] = 0 df.loc[df['ma_fast'] > df['ma_slow'], 'signal'] = 1 df['cross'] = df['signal'].diff() # +1 金叉, -1 死叉

1.2 MACD 指数平滑异同移动平均线

概念：MACD 通过快慢 EMA 的差值（DIF）与信号线（DEA）的交叉，以及柱状图的变化，同时捕捉趋势方向和强度。

\[ \text{DIF} = \text{EMA}_{12} - \text{EMA}_{26} \] \[ \text{DEA} = \text{EMA}_9(\text{DIF}) \] \[ \text{MACD柱} = 2 \times (\text{DIF} - \text{DEA}) \]

原理 DIF 上穿 DEA 为金叉（零轴上更强），DIF 下穿为死叉。柱状图由绿转红（负转正）预示多头启动。背离是重要信号：价格新高但 DIF 不创新高 = 顶背离（看跌）。

def calc_macd(close, fast=12, slow=26, signal=9): ema_fast = close.ewm(span=fast, adjust=False).mean() ema_slow = close.ewm(span=slow, adjust=False).mean() dif = ema_fast - ema_slow dea = dif.ewm(span=signal, adjust=False).mean() macd_bar = 2 * (dif - dea) return dif, dea, macd_bar

1.3 RSI 相对强弱指标

概念：RSI 衡量价格变动速度和幅度，值域 0–100。用于判断超买超卖和趋势反转。

\[ \text{RS} = \frac{\text{AvgGain}_n}{\text{AvgLoss}_n} \] \[ \text{RSI} = 100 - \frac{100}{1 + \text{RS}} \]

原理 传统阈值：RSI > 70 超买（考虑卖出），RSI < 30 超卖（考虑买入）。Wilder 原始周期 n=14。强趋势中可以用 80/20 或 90/10。RSI 背离比阈值更可靠。

def calc_rsi(close, period=14): delta = close.diff() gain = delta.clip(lower=0) loss = -delta.clip(upper=0) avg_gain = gain.ewm(alpha=1/period, adjust=False).mean() avg_loss = loss.ewm(alpha=1/period, adjust=False).mean() rs = avg_gain / avg_loss rsi = 100 - (100 / (1 + rs)) return rsi

1.4 布林带 Bollinger Bands

概念：以均线为中心，上下各偏移 k 倍标准差形成通道。价格触及边界时预示回归或突破。

\[ \text{Middle} = \text{SMA}_n \] \[ \text{Upper} = \text{Middle} + k \cdot \sigma_n \] \[ \text{Lower} = \text{Middle} - k \cdot \sigma_n \] \[ \%b = \frac{P - \text{Lower}}{\text{Upper} - \text{Lower}} \]

原理 默认 n=20, k=2。价格触及下轨（%b < 0）可能超卖反弹，触及上轨（%b > 1）可能超买回调。带宽收缩（Squeeze）预示大波动即将来临。带宽 = (Upper-Lower)/Middle。

def calc_bollinger(close, period=20, k=2): middle = close.rolling(period).mean() std = close.rolling(period).std() upper = middle + k * std lower = middle - k * std pct_b = (close - lower) / (upper - lower) bandwidth = (upper - lower) / middle return middle, upper, lower, pct_b, bandwidth

1.5 通道突破 Channel Breakout

概念：价格突破过去 N 日的最高价即买入，跌破最低价即卖出。这是海龟交易法的核心进场逻辑。

\[ \text{Signal} = \begin{cases} \text{BUY}, & P_t > \max(P_{t-1}, \dots, P_{t-n}) \\ \text{SELL}, & P_t < \min(P_{t-1}, \dots, P_{t-n}) \end{cases} \]

原理 Donchian 通道用 n=20（短线）或 n=55（中长线）。海龟系统加仓规则：价格每突破 0.5 ATR 加仓一次，最多 4 次。突破入场是最纯粹的动量策略。

def donchian_signal(high, low, close, period=20): upper = high.rolling(period).max().shift(1) lower = low.rolling(period).min().shift(1) buy = close > upper sell = close < lower return buy, sell

二、出场 Exit — 何时卖出？

出场是策略中最被低估的环节——它直接决定了盈亏比。出场分为被动（止损/止盈触发）和主动（信号反转/时间到期）两类。

2.1 固定止盈/止损

以入场价为基准，设定固定百分比或绝对金额的出场点。

\[ P_{\text{SL}} = P_{\text{entry}} \times (1 - r_{\text{stop}}) \] \[ P_{\text{TP}} = P_{\text{entry}} \times (1 + r_{\text{target}}) \]

关键 盈亏比至少 ≥ 2:1。常见设置：止损 -5\%，止盈 +10\% 至 +15\%。

2.2 ATR 动态止损

用 ATR（平均真实波幅）衡量市场波动，适应不同波动环境。

\[ \text{TR} = \max(H-L,\ |H-C_{t-1}|,\ |L-C_{t-1}|) \] \[ \text{ATR}_n = \text{EMA}_n(\text{TR}) \] \[ P_{\text{SL}} = P_{\text{entry}} - k \cdot \text{ATR} \]

关键 k 通常取 2–3。波大止损宽、波小止损窄，比固定百分比更合理。

2.3 移动止损 Trailing Stop

止损线随价格向有利方向移动，回撤到一定程度触发出场。

\[ \text{Stop}_t = \max(\text{Stop}_{t-1},\ P_t \times (1 - r_{\text{trail}})) \]

关键 跟踪比例通常 5\%–15\%。锁利的同时允许正常回调。是趋势跟踪的标准配置。

2.4 信号反转 + 时间退出

信号反转：出现反向进场信号时平仓（如金叉买入 → 死叉卖出）。

时间退出：持仓超过 N 天强制平仓，防止资金长期占用低效持仓。

\[ \text{Exit} = \begin{cases} \text{True}, & \text{signal} = -1 \text{ 或 } t - t_{\text{entry}} > N_{\text{max}} \end{cases} \]

三、仓位 Position — 买多少？

仓位管理决定资金效率和风险暴露。同一信号，仓位不同，最终收益可以差 5–10 倍。核心思想：胜率高多下注，风险大多下注的反面——风险大少下注。

3.1 凯利公式 Kelly Criterion

概念：已知胜率和盈亏比，计算最优下注比例，使长期资本增长率最大化。

\[ f^* = \frac{bp - q}{b} = p - \frac{q}{b} \] \[ \text{其中: } p = \text{胜率},\ q = 1-p,\ b = \text{盈亏比（净盈利/净亏损）} \]

原理 如果 p=0.5, b=2，则 f* = 0.25，即每次下注 25\% 资金最优。实际交易中常使用半凯利（f*/2）以降低回撤，或分数凯利因胜率和盈亏比本质上是估计值。

案例：某策略胜率 45\%，平均盈利 8\%，平均亏损 3\%，则 b = 8/3 ≈ 2.67，f* = 0.45 - 0.55/2.67 ≈ 0.244。若本金 100 万，单笔下注 24.4 万。半凯利则为 12.2 万。
def kelly(win_rate, avg_win, avg_loss): b = avg_win / avg_loss f = win_rate - (1 - win_rate) / b return max(0, f) # 负值表示不应下注 # 示例: 胜率 45%, 平均盈亏比 2.67 f = kelly(0.45, 0.08, 0.03) # 结果: 0.244

3.2 风险平价 Risk Parity

概念：让组合中每个资产贡献相等的风险（波动率），而非相等金额。风险高的资产分配更少资金。

\[ w_i \propto \frac{1}{\sigma_i} \] \[ w_i = \frac{1/\sigma_i}{\sum_{j} 1/\sigma_j} \]
def risk_parity_weights(returns): vols = returns.std() # 每只股票的年化波动率 inv_vols = 1.0 / vols weights = inv_vols / inv_vols.sum() return weights

3.3 波动率目标仓位

概念：设定目标波动率（如年化 20\%），根据当前市场实际波动动态调整仓位。波动高 → 减仓，波动低 → 加仓。

\[ w = \min\left(1, \frac{\sigma_{\text{target}}}{\sigma_{\text{realized}}}\right) \]

原理 假设目标年化波动率 20\%，当前市场实现波动率 40\%，则仓位降至 50\%。这天然实现了"高波减仓、低波加仓"。

def vol_target_position(returns, target_vol=0.20, window=60): realized_vol = returns.rolling(window).std() * np.sqrt(252) weight = np.clip(target_vol / realized_vol, 0, 1.5) return weight

四、风控 Risk — 亏得起多少？

风控不是策略的一部分——它是策略能存活的前提。好的风控回答三个问题：最多亏多少？亏的概率有多大？亏的时候多久能回来？

4.1 VaR 风险价值 & CVaR

概念：VaR 回答"在给定置信度下，未来 N 天最多亏多少"。CVaR（条件风险价值）则进一步回答"一旦突破 VaR，平均会亏多少"。

\[ \text{VaR}_\alpha = -\inf\{ x : P(R \le x) \ge 1-\alpha \} \] \[ \text{CVaR}_\alpha = -\mathbb{E}[R \mid R \le -\text{VaR}_\alpha] \]

含义 95\% 置信度下日 VaR = -2\% 表示：有 95\% 把握单日亏损不超过 2\%，即每 20 个交易日约有 1 天亏损超过 2\%。

def calc_var(returns, confidence=0.95): var = np.percentile(returns, 100 * (1 - confidence)) cvar = returns[returns <= var].mean() return -var, -cvar # 返回正数表示亏损百分比

4.2 最大回撤 Max Drawdown

概念：从历史净值最高点到随后最低点的最大跌幅，是衡量策略"最惨时刻"的核心指标。

\[ \text{MDD} = \max_{t \in [0,T]} \left( \frac{\max_{s \in [0,t]} V_s - V_t}{\max_{s \in [0,t]} V_s} \right) \]

阈值 机构通常设 MDD 硬止损线 15\%–25\%。MDD > 30\% 的策略需要极强的心理承受力和资金冗余。

def max_drawdown(values): peak = values.cummax() dd = (values - peak) / peak return dd.min()

4.3 夏普比率 Sharpe Ratio

概念：衡量每承担单位风险获得的超额回报。是最通用的策略评价指标。

\[ \text{Sharpe} = \frac{R_p - R_f}{\sigma_p} \] \[ \text{年化夏普} = \sqrt{252} \times \frac{\bar{r} - r_f}{\sigma_r} \]

基准 夏普 > 1 算合格，> 2 优秀，> 3 极其罕见（需警惕过拟合）。纯粹 Beta 收益（大盘涨你也涨）的夏普通常 < 0.5。

def sharpe(returns, rf=0.02): excess = returns - rf/252 return np.sqrt(252) * excess.mean() / returns.std()

4.4 Calmar 比率

年化收益率除以最大回撤（取绝对值）。修正夏普只关心波动率的问题——波动率不区分"向上的波动"和"向下的波动"。

\[ \text{Calmar} = \frac{\text{年化收益率}}{|\text{MDD}|} \]

基准 Calmar > 0.5 可接受，> 1 良好。趋势跟踪策略的 Calmar 通常比均值回归策略高。

4.5 分散化 & 相关性控制

单票上限：单一标的仓位不超过总资金的 N\%（通常 10\%–25\%）。

行业上限：同行业暴露不超过 30\%–40\%。

相关性矩阵：定期检查持仓间的相关系数，避免买一堆高度同涨同跌的东西（表面分散 → 实质集中）。

\[ \rho_{ij} = \frac{\text{Cov}(R_i, R_j)}{\sigma_i \sigma_j} \]

小结：四个要素的关系

进场决定机会多少，出场决定盈亏比，仓位决定资金效率，风控决定能不能活到下一轮。

一个完整的量化策略是这四个要素的有机组合。回测时不要只盯着总收益率——要看夏普、最大回撤、Calmar 和胜率/盈亏比的拆解。每一个要素都可以独立优化、对比替换，这正是量化交易相比主观交易最大的优势。
